1. Đảm bảo bạn đã cài đặt Don't Starve Together
2. Cài đặt môi trường phát triển
3. Tạo mod đầu tiên của bạn

## 📝 Ghi chú

- API này dành cho Don't Starve Together
- Một số tính năng có thể khác biệt giữa các phiên bản
